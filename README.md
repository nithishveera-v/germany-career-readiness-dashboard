# Germany Career Readiness Dashboard

A responsive web application built using HTML, CSS, and JavaScript to help users track their preparation for working in Germany.

## Features

- Career Readiness Score Calculator
- Skills Tracker
- German Language Level Tracker
- Savings Goal Tracker
- Job Application Tracker
- Germany Roadmap Checklist
- LocalStorage Data Persistence
- Responsive User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- LocalStorage
