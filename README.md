# Germany Career Readiness Dashboard

A responsive web application built using HTML, CSS, and JavaScript to help users track their preparation for working in Germany. The dashboard calculates career readiness based on technical skills, German language level, savings progress, and job applications.

---

## Features

- Readiness Score Calculator
- Skills Tracker
- German Language Level Selection
- Savings Goal Tracker
- Job Application Tracker
- Germany Roadmap Checklist
- LocalStorage Support
- Responsive User Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- LocalStorage
